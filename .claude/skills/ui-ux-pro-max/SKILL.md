# UI/UX Pro Max Skill

This skill provides advanced UI/UX capabilities for the APOFF-AI application.

## Features

- Design system management
- Component library
- Style guide enforcement
- Accessibility checks
- Responsive design patterns

## Usage

Activate this skill when working on UI components or design-related tasks.

## Data Files

- `data/colors.csv` - Color palette definitions
- `data/icons.csv` - Icon library mappings
- `data/typography.csv` - Typography scale
- `data/stacks/*.csv` - Tech stack component mappings

## Scripts

- `scripts/core.py` - Core design utilities
- `scripts/design_system.py` - Design system management
- `scripts/search.py` - Component search utilities
